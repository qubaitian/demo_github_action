RESTful API¶

POST /quant/indicator/windows_rolling¶

滑动窗口金融指标

Parameters

  • prices – dict 资产净值序列，其中支持多资产批量计算

  • window_days – int, optional 窗口长度，单位天,默认365

  • indicators –

    list 指标列表,支持:

    • std: 滚动标准差(波动率)

    • std_describe: 滚动标准差统计

  • Example:

    {
    "prices":{
        "001001":{
        "2021-09-30":2.1,
        "2021-10-07":2.11,
        ...
        "2021-10-25":2.25,
        "2021-11-20":2.30
        }
    },
    "indicators": ["std","std_describe"],
    "window_days": 365
}

    "code": 0,
    "data": {
        "std": {
            "001001": [

POST /quant/indicator/indicators¶

Support calculation of various financial indicators.

Parameters

  • prices – dict Fund net value sequence

  • benchmark – dict Benchmark net value sequence, optional

  • fill_method –

    str, default ‘None’ Method to use for filling holes in {prices}/{benchmark} Series

    • ffill: propagate last valid observation forward to next valid back

    • fill / bfill: use next valid observation to fill gap.

  • indicators –

    list 金融指标

    • returns: 每日收益序列

    • cum_returns: 每日累计净值序列

    • return_describe: 每日收益统计

    • ret_t: 区间累计收益率

    • ret_a: 区间年化收益率

    • vol_t: 区间波动率

    • vol_a: 区间年化波动率

    • mdd: 区间最大回撤

    • mdd_start: 最大回撤起始索引

    • mdd_end: 最大回撤终止索引

    • mdd_recovery: 最大回撤收复失地日

    • dd_history: 回撤历史

    • mdd_days: 最大回撤天数:从高点到低点的天数

    • mdd_recovery_days: 最大回撤恢复天数

    • after_mdd_return: 最大回撤后到现在的收益

    • mr: 区间最大涨幅

    • mr_start: 最大涨幅起始索引

    • mr_end: 最大涨幅终止索引

    • mr_fall: 最大涨幅重新抹平日

    • mr_days: 最大涨幅天数

    • rise_history: 增长历史

    • mr_falling_days: 最大涨幅回落天数

    • sharpe: 夏普比率

    • calmar: 卡玛比率

    • info: 信息比率

    • sortino: sortino

    • downside_risk: 下行风险

    • annual_downside_risk: 年化下行风险

    • alpha: 阿尔法, 注：需要基准数据

    • beta: beta系数，注：需要基准数据

    • treynor: 特雷诺比率，注：需要基准数据

    • passive_influence_ratio: ，注：需要基准数据

    • active_influence_ratio:，注：需要基准数据

  • periods – list Support D,W,M,Y,2W,7D…thisyear,total

  • Example:

    {
    "prices":{
        "2021-09-30":2.1,
        "2021-10-07":2.11,
        ...
        "2021-10-19":2.21,
        "2021-10-20":2.2
    },
    "benchmark":{
        "2021-09-30":2.1,
        "2021-10-07":2.11,
        ...
        "2021-10-19":2.21,
        "2021-10-20":2.11
        },
    "indicators": ["alpha","returns","cum_returns","alpha","ret_t","ret_a","vol_t","vol_a","mdd","mdd_start","mdd_end","dd_history","mdd_days","mdd_recovery","mdd_recovery_days","mr_start","mr_end","mr_days","rise_history","mr_fall","mr_falling_days","sharpe","calmar","info","sortino","beta","treynor","passive_influence_ratio","active_influence_ratio","downside_risk","annual_downside_risk"],
    "fill_method":"None",
    "periods": ["total"]
}

POST /quant/indicator/simulator¶

蒙特卡洛模拟.

Parameters

  • prices – list Fund net value sequence

  • method – str 预测方法,支持bootstrap

  • T – int, optional 预测长度

  • percentile – list, optional 预测概率分布

  • Example:

    {
    "prices":[2.1,2.11,2.12,2.1,2.11,2.15,2.2,2.21,2.25,2.21,2.24],
    "method":"bootstrap",
    "T":10,

POST /quant/indicator/windows¶

窗口日历金融指标

Parameters

  • prices – dict 资产净值序列，其中支持多资产批量计算

  • window – str 窗口定义，支持D,W,M,Y格式以及7D格式，其中W按每周5划分，M按月底划分

  • indicators –

    list 指标列表,支持:

    • returns: 区间累计收益率

    • winrate: 区间胜率

    • longest_win_period: 最长连续正收益

  • Example:

    {
    "prices":{
        "001001":{
        "2021-09-30":2.1,
        "2021-10-07":2.11,
        ...
        "2021-10-25":2.25,
        "2021-11-20":2.30
        }
    },
    "indicators": ["returns","winrate","longest_win_period"],
    "window": "2W"
}
Returns

json windows的个数会大于returns 1个，windows代表窗口的位置，n个窗口位置夹着n-1个窗口

  • Example:

    {
    "code": 0,
    "data": {
        "longest_win_period": {
            "001001": {
                "end": "2021-11-20",
                "period": 5,
